Card Guessing Game For this project, we were given Java files and needed to code them based on the instructions to make a game. When the game runs a window would pop up showing 2 cards and ask the user whether they think the next card selected would be before, between, or after the ones shown. Depending on if the user guesses right or wrong, they would either win or lose double the amount of money they bet. We were responsible for making the game run by doing things such as storing the user guess, the amount of money they have, the amount they bet, etc.
